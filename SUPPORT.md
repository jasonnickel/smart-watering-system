# Support

## Where To Ask What

- GitHub Discussions: setup help, architecture questions, integration ideas, and operational advice
- GitHub Issues: reproducible bugs, regressions, and well-scoped feature requests
- Security reports: follow `SECURITY.md`

## Before Opening A Bug

- Re-run the command in shadow mode if possible
- Check `journalctl -u smart-water -f`
- Include relevant config context without posting secrets
- Mention whether you are using Ambient Weather, MQTT, n8n, and systemd timers

## Expectations

This is a maintained personal project, not a commercial support channel. Clear reproduction steps and logs will always help more than long descriptions without specifics.
