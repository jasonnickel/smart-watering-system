# Code of Conduct

This project aims to be practical, kind, and technically honest.

## Expected Behavior

- Be respectful and assume good intent.
- Give feedback about code and ideas, not people.
- Prefer clarity over point-scoring.
- Welcome questions from users who are new to irrigation, homelab tooling, Node.js, or GitHub.
- Be honest about uncertainty, limitations, and operational risk.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Dismissive or demeaning language
- Deliberate misinformation about what the software does
- Sharing private credentials, personal data, or security details irresponsibly

## Enforcement

Maintainers may edit, hide, lock, or remove content that makes the project less safe or less useful for others.

For serious concerns, contact the maintainer through the reporting path in `SECURITY.md`.
